# Challenging-Liberal-Democracy-in-Eastern-Europe
Replication code and data for the article "Challenging Liberal Democracy in Eastern Europe: Exploring Electoral Support for Illiberal Parties in the 2023 Slovak Parliamentary Election", accepted for publication in _Europe-Asia Studies_.
The paper examines which voter attitudes that run counter to the core tenets of the liberal democratic order contributed to the electoral success of parties advocating an illiberal backlash. The analyses draw on data from the post-election study of the 2023 Slovak parliamentary election.
